鼠标重量：

- 轻：70-90克重
- 中：100克左右的鼠标
- 重：超过120克



对称鼠标和右手鼠标（左高右低）



微动开关

- DPI越高，手腕移动同样距离，鼠标指针的移动距离越大。办公，设置 3000 DPI 就很灵敏了

- 报告率越高，延迟越小，指针移动越平滑。游戏鼠标需要达到1000Hz的回报率，也就是1毫米响应时间。
