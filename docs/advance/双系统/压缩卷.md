## 硬盘分区时提示“选定进行压缩的卷可能已损坏，请使用Chkdsk修复损坏问题”的解决办法

```cmd
chkdsk c: /f
```

按回车，将检查并修复C盘存在的磁盘错误。

该卷正被另一进程使用，无法运行chkdsk。是否计划在下一次系统重新启动时检查此卷？（Y/N）”，输入“y"后按“回车键”，会提示“将在下一次系统重新启动时检查此卷”。
