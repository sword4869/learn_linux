| 表达式       | 作用                | 示例（路径=`/a/b/c`）                 |
| :----------- | :------------------ | :------------------------------------ |
| `${PWD##*/}` | 提取最后一级目录名  | `c`，同`$(basename "$PWD")`（效率低） |
| `${PWD#*/}`  | 删除第一个 `/` 左侧 | `a/b/c`                               |
| `${PWD%/*}`  | 删除最后一级目录    | `/a/b`                                |
| `${PWD%%/*}` | 删除第一个 `/` 右侧 | ``（空）                              |
