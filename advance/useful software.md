# Beautify

## Desktop

安装
```bash
# Gnome, ubuntu default desktop
sudo apt install ubuntu-desktop

# KDE
sudo apt install kubuntu-desktop
```

卸载
```bash
# Gnome
$ sudo apt purge gnome-shell
```

在开机启动登陆账户前，右上角可以选择要用的桌面类型。
